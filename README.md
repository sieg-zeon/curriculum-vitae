- サーバレスアーキテクチャ
- CloudFormation, Serverless Framework、CDKを使ったIaC
- シェルスクリプトでの自動化
- Dockerでのコンテナ開発

- 人数が一人の時もあったり、少なかったため、責任を持ってissueの管理や実装が行えた。
